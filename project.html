Projects

Women cloth reviews prediction
Link

During my internship at the YBI Foundation, I embarked on an exciting project focused on women's clothing prediction. Leveraging the Multinomial Naive Bayes (MultinomialNB) model, I analyzed extensive datasets to forecast trends and preferences in women's fashion. This involved processing vast amounts of clothing-related data to make insightful predictions about styles, colors, and designs that would resonate with the target audience. My work not only honed my data analysis and machine learning skills but also contributed valuable insights to aid the YBI Foundation in tailoring their initiatives to empower women through fashion choices.

Python ATM Machine
Link

My first-year engineering project centers on creating an ATM (Automated Teller Machine) simulation using Python. This project aims to replicate the functionalities of a real ATM, allowing users to perform common banking tasks such as withdrawing, depositing, and checking account balances. Through Python programming, we are implementing a user-friendly interface, secure authentication methods, and transaction processing. This project not only enhances our coding skills but also provides valuable insights into the world of finance and user interface design. It showcases our ability to apply engineering principles to real-world problems and lays the foundation for future innovative projects in the field.
